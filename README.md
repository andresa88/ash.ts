## Development

First install:
```bash
$ npm i
$ npm run build
```

One time test:
```bash
$ npm run test
```

One time test changes files only:
```bash
$ npm run test:changed
```

Test watch mode:
```bash
$ npm run test:watch
```

Test watch changes files only:
```bash
$ npm run test:watch:changed
```

Coverage report:
```bash
$ npm run test:coverage
```

Build docs:
```bash
$ npm run docs
```

Lint files:
```bash
$ npm run lint
```

Clean everything:
```bash
$ npm run clean
```
